Desenvolvimento Frontend

Basicamente o desenvolvimento frontend consiste na materialização de um layout idealizado por um designer, transformando-o em uma página navegável e interativa, utilizável pelo usuário final. Em resumo, consiste no desenvolvimento daquilo que é "exergado" e será usado pelo usuário.

Desenvolvimento Backend

Atuando "por de trás" dos panos, o desenvolvimento backend é a parte responsável pela análise de uma situação problemática, cuja solução, ou caminho para ela, é obtido através do processamento de dados brutos, que tem como produto final a informação. Em resumo, é a parte responsável por analisar o domínio e Desenvolver uma regra de negócio de forma a validar e contextualizar dados, criando um caminho para [ou sendo a própria] soluçao para um determinado problema.
